Privacy Policy for LUMIS

Last updated: February 18, 2026

LUMIS respects your privacy.

1. Data Collection
LUMIS does not collect or store personal data outside of Discord.
The bot may temporarily process:
- User IDs
- Server IDs
- Channel IDs
- Message content (only when required to execute a command or moderation feature)

2. Data Storage
Any stored data (such as server configuration or moderation settings) is used solely to provide bot functionality.
No data is sold, shared, or used for tracking.

3. Data Removal
Server administrators may request data removal by removing the bot from their server.

4. Third Parties
LUMIS does not share data with third parties.

5. Contact
For any privacy-related concerns, please contact the bot developer.
